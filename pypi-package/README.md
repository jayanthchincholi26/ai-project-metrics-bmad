# ai-metrics-capture

Installs the AI-accelerated engineering metrics capture tooling into a git
repository with one command:

```
uvx ai-metrics-capture install
```

Run from the root of the git repo you want to instrument. This is a third,
additive distribution path alongside the existing manual zip download and
`curl`/`irm` one-line installers — see the main project's `INSTALL.md` for
what gets installed and what to do next.
